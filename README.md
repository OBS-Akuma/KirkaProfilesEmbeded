# Kirka Profile Embed

Dynamic profile cards for Kirka players.

[![Example](https://kirkaprofilesembeded.morgan905499.workers.dev/?meow=NUGGET)](https://kirka.io/profile/NUGGET)

## Usage

Replace NUGGET with any player's short ID:

(https://kirkaprofilesembeded.morgan905499.workers.dev/?meow=NUGGET)
![Profile](https://kirkaprofilesembeded.morgan905499.workers.dev/?meow=NUGGET)

Clickable:

[![Profile](https://kirkaprofilesembeded.morgan905499.workers.dev/?meow=NUGGET)](https://kirka.io/profile/NUGGET)
